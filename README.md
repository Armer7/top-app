Train creat components in Next.js with TypeScript

## Getting Started

First, setup the development server:

```bash
npm install
# or
yarn install
```
Second, run the dev server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
